# Installation 🔧

Make sure to have **NodeJS** and **Expo** installed.

```bash

# Clone repository
git clone https://github.com/EwertonSolitto/portfolio-react-native.git

# Go to the directory
cd portfolio-react-native

# Install dependencies
npm i

# Start Application
npm start

```
